# Desafio: Sorteio de Amigo Secreto

Este projeto consiste em uma aplicação simples para realizar o sorteio de um "amigo secreto". Os usuários podem inserir os nomes de seus amigos em uma lista, e a aplicação selecionará aleatoriamente um nome da lista ao clicar no botão "Sortear Amigo".

## Funcionalidades

- **Adicionar nomes:** O usuário pode adicionar os nomes dos amigos em um campo de texto e clicar no botão "Adicionar" para inserir o nome na lista.
- **Validar entrada:** Se o campo de texto estiver vazio, a aplicação exibirá um alerta pedindo para o usuário inserir um nome válido.
- **Visualizar a lista:** Todos os nomes inseridos pelo usuário serão exibidos em uma lista visível na página.
- **Sorteio aleatório:** Quando o usuário clicar no botão "Sortear Amigo", um nome será selecionado aleatoriamente da lista e será exibido na tela.

## Como utilizar

1. **Inserir nomes:**

   - Digite o nome de um amigo no campo de texto.
   - Clique no botão "Adicionar" para adicionar o nome à lista.

2. **Sorteio:**
   - Após adicionar os nomes, clique no botão "Sortear Amigo".
   - Um nome será selecionado aleatoriamente e será exibido na tela.

## Tecnologias utilizadas

- **HTML:** Para a estruturação da página.
- **CSS:** Para o estilo visual da aplicação.
- **JavaScript:** Para a funcionalidade de adicionar nomes à lista e realizar o sorteio aleatório.

## Exemplo de uso

1. Abra a aplicação.
2. Adicione os nomes de amigos na lista.
3. Clique em "Sortear Amigo" para ver quem será o amigo secreto.

## Observações

- Certifique-se de que o campo de entrada não esteja vazio ao tentar adicionar um nome.
- O sorteio é feito de forma aleatória, e o nome sorteado será exibido na tela imediatamente.
